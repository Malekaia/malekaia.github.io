How to install Rust on UNIX

A guide to compiling your first Rust program

By Malekaia • 02/06/2022 •
Contents:

Installing rust on UNIX systems:

To compile a Rust program you'd need to use the rustc compiler, to install this on a UNIX based system such as Ubuntu using the Curl command:

curl https://sh.rustup.rs -sSf | sh

This downloads and runs the rustup-init.sh script which installs the language for you.

You can also install Rust using the apt package manager:

sudo apt install rustc

Hello World!

Create a file called main.rs and create a new function called main. In the main function's body, use a println! macro to write a "Hello World!" string to the console.

fn main() {
  println!("Hello World!");
}

After saving the file, run the following in your terminal and press enter:

rustc main.rs

When the program has finished compiling, the output is an executable file called main, which we would run using:

./main

Which would give us an output of Hello World!
